WINDOWS:
1. Install CMake and make sure its on your %PATH%. The cmake version must be at least 3.12 due to breaking changes with vs2017 and boost.
2. Install NASM from http://www.nasm.us/. Make sure it's on your %PATH% in the working terminal
3. Install 7-Zip, make sure 7z is on your %PATH% in the working terminal
4. Install Git, make sure git is on your %PATH% in the working terminal
